# Kitchen Ticket Reader Mobile App V3

This version lets you edit the setup inside the app from your phone.

You can add/edit:
- Menu items
- Seasonings / sauces
- Temps / cook styles
- Sides
- Ignore words

The setup saves in the phone/browser using localStorage.

## Files to upload to GitHub Pages

Upload:
- index.html
- manifest.json
- sw.js
- icon.svg
- README.md

Then go to Settings > Pages > Deploy from branch > main / root.

## How to use

1. Open the app.
2. Tap Setup Menu.
3. Add your menu items, seasonings, temps, sides, and ignore words.
4. Tap Save Setup.
5. Go back to Scan Tickets.
6. Take or upload ticket photos.
